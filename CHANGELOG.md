## [1.0.0] - 2025-11-25

### Added

#### Components (26)
- **Button**: Multiple variants (default, secondary, outline, ghost) and sizes
- **Badge**: Status and category badges with 4 variants
- **Input**: Text input with various types support
- **Textarea**: Multi-line text input
- **Checkbox**: Checkbox with indeterminate support
- **RadioGroup**: Radio button group with Radix UI
- **Switch**: Toggle switch component
- **Slider**: Range slider with customizable min/max/step
- **Card**: Container component with shadow-retro style
- **Container & Section**: Layout components
- **Table**: Data table with header, body, and footer
- **Avatar**: User avatar component
- **Skeleton**: Loading placeholder
- **Spinner**: Loading spinner
- **Progress**: Progress bar
- **Tabs**: Tab navigation component
- **Accordion**: Collapsible content sections
- **Alert**: Alert messages with variants (info, success, warning, error)
- **Dialog**: Modal dialog
- **Tooltip**: Hover tooltips
- **Popover**: Popover content
- **DropdownMenu**: Dropdown menu
- **Select**: Select dropdown
- **Typography**: H1, H2, H3, P components
- **CodeBlock**: Code display component
- **ThemeToggle**: Light/dark mode toggle
