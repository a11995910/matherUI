# MatherUI 设计规范 (Style Guide)

MatherUI 是一个受 MotherDuck 启发，采用复古/野兽派风格 (Retro/Brutalist) 的 React 组件库。本规范旨在定义其核心视觉元素和组件使用规则。

## 1. 核心理念 (Core Philosophy)
- **大胆 (Bold)**: 使用高对比度的颜色和粗边框。
- **复古 (Retro)**: 借鉴早期计算机界面的美学，如硬阴影、衬线字体。
- **功能至上 (Functional)**: 界面元素清晰明确，强调内容。

## 2. 色彩系统 (Color System)

### 基础色 (Base Colors)

#### 亮色模式 (Light Mode)
- **背景色 (Background)**: `#fbf9f5` (米白色/纸张色) - 用于页面背景。
- **前景色 (Foreground)**: `#1a1a1a` (近黑色) - 用于主要文本。
- **边框色 (Border)**: `#000000` (纯黑) - 用于所有组件的边框，强调轮廓。

#### 暗色模式 (Dark Mode) ✨ NEW
- **背景色 (Background)**: `#0a0a0a` (深黑色) - 用于页面背景。
- **前景色 (Foreground)**: `#f5f5f5` (浅灰白) - 用于主要文本。
- **边框色 (Border)**: `#ffffff` (纯白) - 用于所有组件的边框。
- **卡片背景 (Card)**: `#141414` - 用于卡片、面板等组件背景。
- **静音背景 (Muted)**: `#1a1a1a` - 用于次要背景。

### 品牌色 (Brand Colors)
- **主色 (Primary)**: 
  - 亮色模式: `#66b3ff` (天蓝色)
  - 暗色模式: `#4d9fff` (亮蓝色)
- **次色 (Secondary)**: 
  - 亮色模式: `#ff8f4d` (橙色)
  - 暗色模式: `#ff6b2c` (亮橙色)

### 辅助色 (Utility Colors)
- **静音色 (Muted)**: 
  - 亮色模式: `#f0eee9`
  - 暗色模式: `#1a1a1a`
- **静音文本 (Muted Foreground)**: 
  - 亮色模式: `#666666`
  - 暗色模式: `#a0a0a0`

## 3. 排版 (Typography)

### 字体家族 (Font Family)
- **标题 (Headings)**: `Merriweather`, serif - 用于 H1-H6，传达经典、权威的感觉。
- **正文 (Body)**: `Inter`, sans-serif - 用于段落、按钮、输入框，确保可读性。
- **代码 (Code)**: `Monospace` (系统默认) - 用于代码块。

### 阴影 (Shadows)
- **复古阴影 (Retro Shadow)**: `4px 4px 0px 0px #000000` - 用于卡片、按钮常态。
- **悬停阴影 (Hover Shadow)**: `2px 2px 0px 0px #000000` - 用于按钮点击/悬停反馈。

## 4. 组件规范 (Component Specs)

### 基础组件

#### 按钮 (Button)
- **形状**: 矩形，无圆角或极小圆角。
- **边框**: 2px 纯黑边框（暗色模式为白色）。
- **交互**: 悬停时位移 + 阴影变化，产生"按下"的物理感。
- **变体**: Default, Secondary, Outline, Ghost, Link

#### 输入框 (Input)
- **边框**: 2px 纯黑边框（暗色模式为白色）。
- **焦点**: 聚焦时边框变色或阴影加深。
- **高度**: 统一 h-10 (40px)
- **背景**: 使用 `bg-background` 自动适配主题

#### 卡片 (Card)
- **背景**: 使用 `bg-card` 自动适配主题（亮色为白色，暗色为深灰）
- **边框**: 2px 纯黑边框（暗色模式为白色）
- **阴影**: 必须带有复古硬阴影。

### 数据展示组件

#### 表格 (Table)
- **边框**: 所有单元格都有 2px border
- **表头**: 粗体字体，背景色区分
- **背景**: 表格默认透明，单元格使用主题色

#### 徽章 (Badge)
- **尺寸**: 紧凑，适合内联显示
- **边框**: 2px 黑色边框（暗色模式为白色）

#### 头像 (Avatar)
- **形状**: 方形（非圆形）
- **边框**: 2px 黑色边框（暗色模式为白色）

### 反馈组件

#### 弹窗 (Dialog)
- **背景遮罩**: 半透明背景
- **内容**: 使用 `bg-background` + 硬阴影
- **关闭按钮**: 右上角 X 图标

#### 提示框 (Alert)
- **变体**: Info, Success, Warning, Error
- **图标**: 左侧状态图标
- **可关闭**: 可选的关闭按钮
- **背景**: 根据状态使用不同的浅色背景

### 导航组件

#### 导航栏 (Navbar) ✨ NEW
- **位置**: Sticky 粘性定位在顶部
- **高度**: 固定 80px (h-20)
- **Logo**: 左侧，可点击返回首页
- **导航链接**: 
  - 使用 React Router `<Link>` 组件
  - 悬停时改变颜色（`hover:text-primary`）
  - 避免使用锚点链接（`#`），除非是页面内滚动
- **主题切换器**: 右侧显示，日/月图标
- **Get Started 按钮**: 链接到文档页面

#### 底部导航 (Bottom Navigation) ✨ NEW
- **位置**: Fixed 固定在底部中央
- **样式**: 带边框和阴影的容器
- **按钮**: 使用 `Button` 组件配合 `Link`
- **图标**: 每个按钮都有对应图标

#### 下拉菜单 (DropdownMenu)
- **触发器**: 按钮或其他可点击元素
- **内容**: 使用 `bg-background` + 黑色边框
- **项目**: 悬停高亮

#### 标签页 (Tabs)
- **激活状态**: 明显的边框和背景色变化
- **内容**: 带边框的内容区域

### 表单组件

#### 复选框 (Checkbox)
- **形状**: 方形
- **背景**: 亮色模式白色，暗色模式透明
- **选中**: 蓝色背景 + 白色对勾

#### 单选按钮 (Radio)
- **形状**: 方形（非圆形）
- **背景**: 亮色模式白色，暗色模式透明
- **选中**: 内部圆点

#### 开关 (Switch)
- **形状**: 矩形轨道
- **滑块**: 方形滑块

#### 下拉选择 (Select)
- **触发器**: 类似输入框
- **下拉内容**: 带边框列表，使用 `bg-background`

#### 滑块 (Slider)
- **轨道**: 2px 边框矩形
- **滑块**: 方形带硬阴影

#### 进度条 (Progress)
- **容器**: 2px 边框矩形
- **填充**: 主色填充

### 加载组件

#### 骨架屏 (Skeleton)
- **形状**: 矩形块
- **动画**: 脉冲动画
- **边框**: 2px 黑色边框（暗色模式为白色）

#### 加载动画 (Spinner)
- **形状**: 圆形边框
- **动画**: 旋转动画

### 数据可视化

#### 图表 (Charts)
- **容器背景**: 使用 `bg-card` 自动适配主题
- **网格**: 2px 黑色网格线
- **坐标轴**: 粗黑色线条
- **柱状图**: 蓝色填充 + 黑色边框
- **折线图**: 粗线条 + 方形数据点

### 主题切换 (Theme Toggle) ✨ NEW
- **位置**: 导航栏右侧
- **样式**: Ghost 按钮变体
- **图标**: 
  - 亮色模式显示太阳图标
  - 暗色模式显示月亮图标
  - 使用 CSS transition 实现平滑过渡
- **持久化**: 主题偏好保存到 localStorage

## 5. 布局 (Layout)
- **容器 (Container)**: 居中，最大宽度限制，两侧留白。
- **间距 (Spacing)**: 使用宽松的间距，避免元素过于拥挤。
- **分区 (Section)**: 使用边框分隔不同区域。

## 6. 动画和过渡
- **时长**: 快速过渡 (200ms)
- **缓动**: 线性或ease-out
- **状态变化**: 明显的视觉反馈
- **主题切换**: 平滑的颜色过渡

## 7. 交互规范 ✨ NEW

### 导航链接
- **页面跳转**: 统一使用 React Router 的 `<Link>` 组件
- **页面内滚动**: 使用锚点链接 `<a href="#section">`
- **外部链接**: 添加 `target="_blank"` 和 `rel="noopener noreferrer"`

### 按钮链接
- **包裹方式**: 用 `<Link>` 包裹 `<Button>` 组件
```tsx
<Link to="/docs">
  <Button>查看文档</Button>
</Link>
```

### 颜色使用规范
- **避免硬编码**: 不使用 `bg-white` 等固定颜色
- **使用主题色**: 统一使用 `bg-card`, `bg-background`, `text-foreground` 等
- **自动适配**: 所有组件应在亮色/暗色模式下都能正常显示
